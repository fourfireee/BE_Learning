# 写作和示例要求

- 用尽量通俗易懂的语言来精简地描述概念和原理。
- 必要的时候提供 md 文档内的 code snippet 示例或者 mermaid 流程图。
- 必要的时候提供独立的示例代码 folder（在 `BE/examples/NN-*/` 下，每个示例一个 folder，文档里给运行命令）。
- 代码里的关键点，需要在代码里增加注释，用同样的行文风格进行说明。
- md 文件的排版尽量简洁，除了标题，就用 bullet points，不要随意加粗或者斜体。
- 下面每个主要知识点都作为一个独立 md 文件，并在这个 `root.md` 中引用。

# 这套教程的定位

- 这套 `BE/` 教程讲“服务端本身”：网络协议、框架、数据存储、架构集群、上线运维，以及把它们串起来的实战。
- Java 语言基础另有一套 `../Java/` 教程（class、泛型、集合、并发、构建等），本教程默认你已具备或会同步补，不再重复语言语法。
- 语言策略：概念部分尽量语言无关、讲清原理；框架与代码示例以 Java / Spring Boot 为主线，Python / FastAPI 作对照（贴合主力 Java 场景，同时照顾你已有的 Python AIGC 链路）。
- 风格：动手为主。架构、集群、网关这类不只讲概念，会配可运行的最小示例（如 Nginx 网关 + 多实例 + Redis），让抽象的“分布式”看得见、摸得着。
- 顺序建议：00 → Part 0 → Part 5 打基础，Part 6 三个实战分别对应你的三个目标场景，可在学完相关基础后随时跳读。
- 结合你已有的 C++/native/图形/AIGC 背景做类比，帮助快速建立直觉。

# 学习大纲

## Part 0 · 总览与心智转变

0. [本地开发与日常工作流](00-本地开发与日常工作流.md)：先建立后端本地工作方式——应用进程、配置、依赖服务、数据库迁移、curl 调试、日志排查怎么配合。
1. [服务端开发全景](01-服务端全景.md)：从 native/客户端到服务端的思维转变——长期运行的进程、并发处理大量请求、无状态、横向扩展。用“游戏引擎主循环 vs 服务端请求循环”“单机 vs 集群”建立直觉。

## Part 1 · 网络与协议基础

2. [网络协议与连接形式](02-网络协议与连接.md)：TCP/UDP、HTTP/1.1·2·3、HTTPS/TLS、长连接 vs 短连接、WebSocket、SSE、gRPC 各自解决什么、怎么选。
3. [HTTP API 设计](03-http-api设计.md)：REST 风格、状态码、幂等性、版本化、分页、错误约定、OpenAPI 契约。
4. [数据传输与大文件](04-数据传输与大文件.md)：JSON / Protobuf、二进制流、大文件上传下载、分块与断点续传、签名 URL。

## Part 2 · 服务端框架与编程模型

5. [Java 服务端框架（Spring Boot）](05-spring-boot.md)：Controller/Service/Repository 分层、依赖注入、配置、拦截器/过滤器。配示例 `examples/05-spring-boot-rest`。
6. [Python 服务端框架（FastAPI）](06-fastapi.md)：async、pydantic、依赖注入，对比 Flask/Django；何时选 Python 而非 Java。配示例 `examples/06-fastapi-rest`。
7. [同步 vs 异步与并发模型](07-同步异步与并发.md)：线程池 vs 事件循环、阻塞 vs 非阻塞 IO、async/await。
8. [服务端编排与任务流](08-服务端编排与任务流.md)：在后端串联多个下游服务、分支执行、超时/重试/并发限制。配示例 `examples/08-orchestration`。

## Part 3 · 数据与存储

9. [关系型数据库](09-关系型数据库.md)：SQL、表与索引设计、事务、连接池、ORM 与选型。
10. [NoSQL 与缓存](10-nosql与缓存.md)：Redis（缓存/分布式锁/简单队列）、KV/文档库、缓存策略与一致性。
11. [对象存储与 CDN](11-对象存储与cdn.md)：存大体积素材、元数据库 + 对象存储分工、签名 URL 与分发。
12. [消息队列与异步任务](12-消息队列.md)：Kafka/RabbitMQ/任务队列，解耦、削峰、异步生成与回调。

## Part 4 · 架构 · 集群 · 网关

13. [服务拆分与架构演进](13-服务拆分与架构演进.md)：单体 → 服务化 → 微服务，何时该拆、拆的代价。
14. [集群、负载均衡与网关](14-集群负载均衡与网关.md)：多实例如何组织、网关职责（路由/鉴权/限流/聚合）、服务发现、配置中心。配示例 `examples/14-gateway-cluster`。
15. [鉴权与安全](15-鉴权与安全.md)：认证 vs 授权、Session/JWT/OAuth2、API key、限流防刷、密钥管理。

## Part 5 · 工程化 · 上线 · 可观测

16. [配置与环境管理](16-配置与环境管理.md)：多环境、配置注入、密钥不入库。
17. [容器化与部署发布](17-容器化与部署发布.md)：Docker 镜像、CI/CD、发布策略（滚动/蓝绿/金丝雀）、回滚。
18. [测试策略](18-测试策略.md)：单元/集成/契约/E2E、mock 下游、测试数据库。
19. [可观测性与线上排障](19-可观测性与排障.md)：日志、指标(metrics)、链路追踪(tracing)、告警、容量与性能。

20. [实战 A · 短视频特效云端存储与下发](20-实战A-素材存储与下发.md)：对象存储 + 元数据库 + 签名 URL + CDN。配示例 `examples/20-effect-delivery`。
